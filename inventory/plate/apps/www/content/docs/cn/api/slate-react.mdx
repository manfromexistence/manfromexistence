## Transforms

### focusEditorEdge

在指定的边缘（开始或结束）聚焦编辑器。

<APIParameters>
  <APIItem name="editor" type="TEditor">
    编辑器实例。
  </APIItem>
  <APIItem name="options" type="object" optional>
    <APISubList>
      <APISubListItem parent="options" name="edge" type="'start' | 'end'" optional>
        要聚焦的边缘。
        - 'start': 在编辑器开始处聚焦。
        - 'end': 在编辑器结束处聚焦。
        默认值为 'start'。
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

### setNode

在编辑器中设置特定节点的属性。

<APIParameters>
  <APIItem name="editor" type="TEditor">
    编辑器实例。
  </APIItem>
  <APIItem name="node" type="NodeOf<E>">
    要更新的节点。
  </APIItem>
  <APIItem name="props" type="Partial<TNodeProps<N>>">
    要在节点上设置的属性。
  </APIItem>
  <APIItem name="options" type="Omit<SetNodesOptions<E>, 'at'>" optional>
    设置节点的选项，不包括 'at' 属性。
  </APIItem>
</APIParameters>

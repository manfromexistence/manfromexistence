拖放控制器（[name]）

该类被用于提供一个拖放交互。

进口

[name] 是一个附加组件，必须显式导入。 See [link:#manual/introduction/Installation Installation / Addons].

import { DragControls } from 'three/addons/controls/DragControls.js';

代码示例

const controls = new DragControls( objects, camera, renderer.domElement ); // add event listener to highlight dragged objects controls.addEventListener( 'dragstart', function ( event ) { event.object.material.emissive.set( 0xaaaaaa ); } ); controls.addEventListener( 'dragend', function ( event ) { event.object.material.emissive.set( 0x000000 ); } );

例子

[example:misc_controls_drag misc / controls / drag ]

构造函数

[name]( [param:Array objects], [param:Camera camera], [param:HTMLDOMElement domElement] )

[page:Array objects]: 一组可被拖拽的3D Objects。

[page:Camera camera]: 渲染场景的摄像机。

[page:HTMLDOMElement domElement]: 用于事件监听的HTML元素。

创建一个新的 [name] 实例。

事件

dragstart

当用户开始拖拽3D Objects时触发。

drag

当用户拖拽3D Objects时触发。

dragend

当用户开始完成3D Objects时触发。

hoveron

当指针移动到一个3D Object或者其某个子级上时触发。

hoveroff

当指针移出一个3D Object时触发。

属性

[property:Boolean enabled]

是否启用控制器。

[property:Boolean transformGroup]

当[page:DragControls.objects]数组包含一个单个可拖拽的组对象时该选项生效。如果设置为`true`，[name]会转换整个组对象，而不对单个对象做转换。默认值为`false`。

方法

共有方法请参见其基类[page:EventDispatcher]。

[method:undefined activate] ()

添加控制器的事件监听。

[method:undefined deactivate] ()

移除控制器的事件监听。

[method:undefined dispose] ()

若不再需要该控制器，则应当调用此函数。

[method:Array getObjects] ()

返回可拖拽的对象数组。

[method:Raycaster getRaycaster] ()

返回用于相交测试的内部[page:Raycaster]实例。

## 功能

- 将编辑器限制为单个文本块。适用于简短内容或标题。

</PackageInfo>

## 安装

```bash
npm install @udecode/plate-break
```

## 用法

```tsx
import { SingleLinePlugin } from '@udecode/plate-break/react';

const plugins = [
  // ...otherPlugins
  SingleLinePlugin,
];
```

<PackageInfo>

## 功能

- 提供智能自动完成功能，用于提及用户或应用标签。
- 使用 `@` 或 `#` 调用组合框并选择建议。

</PackageInfo>

## 安装

```bash
npm install @udecode/plate-mention
```

## 用法

```tsx
import { MentionPlugin } from '@udecode/plate-mention/react';

const plugins = [
  // ...otherPlugins,
  MentionPlugin,
];
```

## 插件

### MentionPlugin

继承 [TriggerComboboxPlugin](/docs/combobox#options)

<APIOptions>
<APIItem name="insertSpaceAfterMention" type="boolean" optional>
  是否在提及后插入空格。
</APIItem>
</APIOptions>

### MentionInputPlugin

提及输入功能的插件。

## API

### editor.api.insert.mention

重写此 API 方法以自定义提及的插入方式。

### getMentionOnSelectItem

获取提及组合框中选择项目的处理程序。

<APIParameters>
<APIItem name="options" type="object" optional>
<APISubList>
<APISubListItem parent="options" name="key" type="string" optional>
提及插件的插件键。

- **默认值：** `MentionPlugin.key`

</APISubListItem>
</APISubList>
</APIItem>
</APIParameters>

<APIReturns>
  <APIItem type="MentionOnSelectItem">
    提及组合框中选择项目的处理程序。
  </APIItem>
</APIReturns>

## Types

### QueryNodeOptions

用于查询节点entry的接口。

<APIAttributes>
<APIItem
  name="filter"
  type="(entry: NodeEntry) => boolean"
  optional
>

用于过滤节点entry的函数。

- 该函数应接收一个节点entry作为参数并返回一个布尔值。

</APIItem>
<APIItem name="allow" type="string[] | string | null" optional>

有效类型列表。

- 如果为空或未定义，则允许所有类型。

</APIItem>
<APIItem name="exclude" type="string[] | string | null" optional>

无效类型列表。

</APIItem>
<APIItem name="level" type="number[] | number | null" optional>

有效的路径层级。

</APIItem>
<APIItem name="maxLevel" type="number | null" optional>

高于此层级的路径无效。

</APIItem>
</APIAttributes>

### findDescendant

遍历编辑器中的所有节点并返回第一个匹配项。如果没有找到匹配项，则返回 undefined。

<APIParameters>
  <APIItem name="editor" type="Editor">
    要搜索后代节点的编辑器。
  </APIItem>
  <APIItem name="options" type="FindNodeOptions">
    查找节点的选项。它可以包含匹配条件、起始位置、是否反向遍历以及是否包含空节点。
  </APIItem>
</APIParameters>
<APIReturns>
  返回第一个匹配条件的节点entry，如果没有找到匹配项则返回 `undefined`。
</APIReturns>

### getFragmentProp

从节点片段中检索一个一致的属性值。

<APIParameters>
  <APIItem name="fragment" type="TElement[]">
    要搜索属性的节点片段。
  </APIItem>
  <APIItem name="options" type="GetFragmentPropOptions" optional>
    <APISubList>
      <APISubListItem parent="options" name="key" type="string" optional>
        从每个节点提取的属性键。
      </APISubListItem>
      <APISubListItem parent="options" name="defaultValue" type="string" optional>
        如果未找到有效属性时返回的默认值。
      </APISubListItem>
      <APISubListItem parent="options" name="getProp" type="(node: TElement | TText) => any" optional>
        从节点提取属性值的自定义函数。
      </APISubListItem>
      <APISubListItem parent="options" name="mode" type="'all' | 'block' | 'text'" optional>
        确定如何遍历片段：
        - 'all'：检查块节点和文本节点
        - 'block'：仅检查块节点
        - 'text'：仅检查文本节点
        默认值为 'block'。
      </APISubListItem>
    </APISubList>
  </APIItem>
</APIParameters>

<APIReturns>
  在片段中找到的一致属性值，如果没有找到一致的值则返回 undefined。
</APIReturns>

  </APIItem>
  <APIItem name="at" type="Location">

    定义如何执行搜索的选项。

</APIReturns>

<APIItem name="options" type="object">

<APISubList>

<APISubListItem parent="options" name="at" type="Location | null" optional>

- **默认值：** 选区。

</APISubListItem>
<APISubListItem parent="options" name="focus" type="boolean" optional>
如果为 true，返回焦点。否则，返回锚点。
</APISubListItem>

</APISubList>

</APIItem>

</APIParameters>
<APIReturns>

返回位置的点。

</APIReturns>

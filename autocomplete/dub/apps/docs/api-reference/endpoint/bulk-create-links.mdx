og:title: 'Bulk create links with the Dub.co API - API Reference'
---

<Warning>
  Bulk link creation does not support the following features: [Link
  cloaking](https://dub.co/help/article/link-cloaking),
  [Custom social media cards](https://dub.co/help/article/custom-social-media-cards)
</Warning>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.dub.co/links/bulk?workspaceId={WORKSPACE_ID} \
  --header 'Authorization: Bearer {YOUR_TOKEN}' \
  --header 'Content-Type: application/json' \
  --data '[
    {
      "domain": "yourdomain.com",
      "url": "https://www.youtube.com/"
    },
    {
      "domain": "yourdomain.com",
      "url": "https://www.google.com/"
    }
    ...
  ]'
```
```javascript JavaScript
const options = {
  method: 'POST',
  headers: {Authorization: 'Bearer {YOUR_TOKEN}', 'Content-Type': 'application/json'},
  body: JSON.stringify([
    {
      "domain": "yourdomain.com",
      "url": "https://www.youtube.com/"
    },
    {
      "domain": "yourdomain.com",
      "url": "https://www.google.com/"
    },
    ...
  ])
};

fetch('https://api.dub.co/links/bulk?workspaceId={WORKSPACE_ID}', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```
</RequestExample>
